# XRouter
Um roteador meio lixo feito com ESP32 para um trabalho de Fisica.

## Sobre o projeto

Fisicamente, utilizamos uma LED azul com um resistor de 110ohms no pino 4, um botão no pino 23, 3.3v e um interruptor entre o Vin e a fonte de energia.

O botão do pino 23 serve para ativar a debugação (Menu web, Serial) e portanto nao deve ser clicado geralmente. Isso só é checado no boot.
